# Simple steps to run this project

## Clone

```bash
git clone https://github.com/baseergroot/FastApi-Todo-App.git
```
## Install dependencies
```bash
uv sync
```

# Run

```bash
uv run fastapi dev
```

## Visit to see your project
```
http://127.0.0.1:8000
```

## For Documents Visit
```
http://127.0.0.1:8000/docs
```
